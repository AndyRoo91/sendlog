# Design mockup (archive)

The original hi-fi mockup that drove the V1 redesign. It's a standalone
React-via-CDN prototype — **not** part of the app build.

Open `index.html` directly in a browser to scroll through the design canvas
("V1 · Tick Sheet — deep dive" is the canonical reference). See `../../SPEC.md`
for the written handoff spec.

The shipped implementation lives in `frontend/src/ui/` (components) and the
`/design` route (showcase); these files are kept only as a visual reference.
